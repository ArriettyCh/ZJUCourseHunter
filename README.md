# ZJU Course Hunter

## Quick Start

1. 无需配置环境，直接运行 `main.py`，脚本将会自动检测环境，并安装缺少的依赖。如果安装失败，可能是网络问题，或者 python 版本过低（要求 python ≥ 3.9）
2. 首次登录时，需要输入学号和密码，学号和密码将会自动保存
3. 等待选课界面弹出，点击“抢课”并确认，之后浏览器页面将会自动关闭，抢课开始
