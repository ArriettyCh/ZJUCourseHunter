# ZJU Course Hunter



先安装 `requirements.txt`，然后 `playwright install`，这样环境就配好了。

然后运行 `main.py`，首次运行需要输入账号和密码，然后脚本会自动打开教务系统选课界面，然后点击红色的“抢课”按钮并确认即可（页面将会自动关闭并开始抢课）
